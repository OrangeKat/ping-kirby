Exercise

This time, no need to code, just click on the green "Run" button to run your program. It should print "Hello Kirby!" on the terminal output.

Let's jump in the details!

Let's go over this code line by line:

public class Main {

Line 1: This line declares a public class named Main. This is not important for now, we will talk about classes and objects later.

public static void main(String[] args) {

Line 2: This line declares a function, and not any function, the main function! This is the entry point of a Java program. We will see what these keyword mean in the next lesson.

System.out.println("Hello Kirby!");

Line 3: System is a pre-defined class provided by Java, containing some useful methods. out is a static variable within System that represents the output of your program; println is a method of out that can be used to print a line: here, we are printing "Hello Kirby!".
